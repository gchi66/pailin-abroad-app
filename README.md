# Pailin Abroad App

Cross-platform mobile app for **Pailin Abroad** — an English-learning experience built around real-life conversations and practical lessons (EN/TH).

This repo contains the **React Native (Expo)** client app.

## Tech Stack
- **React Native** + **Expo**
- **TypeScript**
- **Supabase** (auth + database + storage)
- **Stripe** (subscriptions / payments — web flow or native, TBD)

## Getting Started

### Prerequisites
- Node (LTS)
- Watchman (recommended)
- Xcode (for iOS simulator)

### Install & Run
```bash
npm install
npm start
